    Module dlangui.core.editable

    This module contains implementation of editable text content.

    Synopsis

    import dlangui.core.editable;

    Functions

    Name Description
    concatDStrings concat strings from array using delimiter
    replaceEolsWithSpaces replace end of lines with spaces
    splitDString split dstring by delimiters
    tokenCategory extracts token category, clearing subcategory
